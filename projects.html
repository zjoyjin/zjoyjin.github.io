Social Media Depression Detection

A research project conducted at Stanford University focusing on human-computer interaction in AI. Built machine learning models using RoBERTa and Decision Trees to detect depression signals in social media content, achieving 82% accuracy. Deployed interactive prototypes using Next.js and Vercel with integrated ML backends.

Tech Stack: Python, RoBERTa, PyTorch, Next.js, Vercel

Research Project - Stanford University
