# Photo Gallery Images

## How to Add Your Photos

1. **Add your photos to this folder** (`images/gallery/`)
2. **Use descriptive filenames** (e.g., `hiking-trip.jpg`, `coffee-cafe.jpg`)
3. **Recommended specs:**
   - Format: JPG or PNG
   - Size: 800x800px (square)
   - File size: Under 500KB for web performance
   - Aspect ratio: 1:1 (square crops work best)

4. **Update the gallery in index.html:**
   - Find the `<!-- Photo Gallery -->` section
   - Add/remove `<div class="gallery-photo">` blocks
   - Change the `src` to your filename

## Placeholder Images

The gallery currently uses placeholder images. Replace them with your own photos!

Example filenames to use:
- `photo-1.jpg`
- `photo-2.jpg`
- `photo-3.jpg`
- etc.

## Image Optimization Tips

- Use tools like [TinyPNG](https://tinypng.com) to compress images
- Crop to square before uploading
- Keep images under 500KB for fast loading
